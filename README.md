## 📌 Overview

This repository contains a comprehensive Exploratory Data Analysis (EDA) of a detailed Netflix dataset covering movies and TV shows up to 2025. The analysis focuses on cleaning, visualizing, and extracting meaningful insights about content trends, ratings, genres, release patterns, and geographic distributions.

## 🎯 Key Objectives

* **Data Cleaning**: Handle missing values, standardize formats, and prepare data for analysis.
* **Trend Analysis**: Uncover temporal patterns in content release and popularity.
* **Genre Exploration**: Identify dominant and emerging content categories.
* **Rating & Popularity**: Investigate relationships between audience ratings, vote counts, and popularity metrics.
* **Geographic Patterns**: Analyze content distribution by country of origin and language.


## 🛠️ Technologies Used

- **Python 3.8+**: Core programming language.
- **Pandas & NumPy**: Data manipulation and numerical operations.
- **Matplotlib & Seaborn**: Static data visualizations.
- **Plotly (optional)**: Interactive visualizations.
- **Jupyter Notebook**: Interactive development environment.

## 🔍 Analysis Highlights

The notebook explores various aspects of the dataset, including:

1. **Overview & Data Cleaning**:
   * Initial data inspection (`info()`, `describe()`, `head()`).
   * Handling missing values.
   * Data type conversions (e.g., dates, numeric columns).
   * Feature engineering (e.g., profit, profit margin).

2. **Univariate Analysis**:
   * Distribution of movie/TV show types.
   * Content ratings and language frequency.
   * Release year trends.
   * Popularity and vote count distributions.

3. **Bivariate & Multivariate Analysis**:
   * **Popularity vs. Rating**: Scatter plots to see if popular content is rated higher.
   * **Budget vs. Revenue**: Relationship and profitability analysis.
   * **Genre & Rating**: Box plots to compare ratings across different genres.
   * **Country of Origin**: Analysis of content production by country.

4. **Time Series & Trend Analysis**:
   * Number of releases per year.
   * Evolution of genre popularity over time.
   * Seasonal or yearly patterns in content additions.

5. **Text Analysis (Optional)**:
   * Word clouds from content descriptions.
   * Keyword extraction from titles or descriptions.
